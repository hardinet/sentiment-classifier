#  Sentiment Analysis IA

Ce projet est un classificateur de sentiments utilisant le Deep Learning (**Transformers DistilBERT**). Il permet d'analyser si un texte est positif ou négatif en quelques secondes.

##  Fonctionnalités
- **Analyse NLP** : Utilisation de modèles de pointe via Hugging Face.
- **Interface Web** : Démo interactive avec Gradio.
- **Prêt pour le déploiement** : Compatible avec Hugging Face Spaces.

## Installation locale
```bash
pip install -r requirements.txt
python app.py
